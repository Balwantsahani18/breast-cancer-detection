# 🎗 Breast Cancer Detection System

An AI-powered web application that predicts breast cancer risk using machine learning.

---

## 🚀 Features

- 🔍 Predict cancer risk using ML model
- 📊 Probability visualization (Benign vs Malignant)
- 📄 PDF report download
- 📤 CSV upload support (optional)
- 🎨 Modern medical UI (React + Vite)
- ⚡ FastAPI backend

---

## 🧠 Tech Stack

### Frontend
- React (Vite)
- Axios
- CSS (Custom UI)

### Backend
- FastAPI
- Scikit-learn
- Pandas / NumPy
- Joblib
